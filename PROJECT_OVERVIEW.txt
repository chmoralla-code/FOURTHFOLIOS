QUICK START (10 MINUTES)
================================================================================

1. RUN LOCALLY:
   cd /workspaces/arch-portfolio
   npm run dev

2. CREATE SUPABASE PROJECT:
   https://supabase.com

3. GET YOUR API KEYS:
   Settings > API

4. UPDATE .env.local:
   NEXT_PUBLIC_SUPABASE_URL=your-url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your-key
   SUPABASE_SERVICE_ROLE_KEY=your-key

5. SET UP DATABASE:
   - Open supabase.sql
   - Paste into Supabase SQL Editor
   - Run it
   - Create storage bucket "project-images"

6. TEST LOCALLY:
   npm run dev
   http://localhost:3000 (public)
   http://localhost:3000/admin (admin)

7. DEPLOY TO VERCEL:
   - Push to GitHub
   - Go to vercel.com
   - Import repository
   - Add env variables
   - Deploy
   - LIVE! 🎉
